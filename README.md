# Daily Planner Bot

## Descrizione:
Un bot di telegram dedicato alla gestione di eventi e promemoria.
Il bot avrà diverse funzionalità, tra cui:
- Gestione (aggiunta, rimozione, modifica) eventi e promemoria;
- Scheduling eventi (ricorrenti o una tantum) e gestione dei conflitti tra eventi;
- Promemoria automatici;
- I/O su file;
- Recap impegni del giorno.
